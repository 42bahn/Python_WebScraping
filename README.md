## Module & Package

    모듈이란 각종 변수, 함수, 클래스를 담고있는 파일
    특정 기능을 .py 확장자로, 파이썬 프로그래밍 언어 형식으로 작성된 파일로 작성

    이러한 모듈을 여러 개 모아놓은 것을 패키지라 한다.

    ex)

    import 모듈이름 [as 별칭]
    import re
    import requests as req
    
    from 키워드를 통해 모듈안의 일부인 변수, 함수, 클래스를 가져올 수 있다.

    ex)
    from selenium import webdriver
